# Comparison of Statistical Methods to Characterize Yield Response to Phosphorus

This repository contains R code used for a simulation study comparing statistical methods to classify phosphorus responsiveness in wheat from soil test phosphorus data.

## Methods compared

- Linear-plateau
- Linear-plateau + ANOVA
- Logistic regression

## What the code does

- Simulates yield response across phosphorus fertilizer rates
- Fits the three classification methods
- Evaluates model performance across simulation scenarios
- Summarizes results with tables and figures

## Purpose

The goal is to compare how well different methods identify responsive and non-responsive site-years under controlled simulation settings.


## Output

The code generates performance metrics such as accuracy and precision, along with summary plots for method comparison.
